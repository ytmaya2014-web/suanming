### 1.3 八字喜忌配戴核心原则

> **最重要的原则：用神（喜神）决定你戴什么，不是缺什么补什么！**
>
> 八字五行全都有，只是力量强弱不同。戴水晶补的是"对你有帮助的五行"（用神/喜神），而不是"五行缺的"。缺的不一定需要补，旺的反而要泄。

| 八字喜 | 首选水晶 | 次选 | 忌戴（忌神五行）|
|--------|----------|------|----------------|
| 喜火 | 石榴石、红发晶、草莓晶 | 红纹石、红碧玺、紫锂辉石 | 水性水晶（黑曜石、茶晶等）|
| 喜木 | 绿幽灵、绿发晶 | 绿松石、紫水晶、孔雀石 | 金性水晶（白水晶、月光石等）|
| 喜水 | 海蓝宝、茶水晶 | 蓝发晶、黑曜石、青金石 | 土性水晶（黄水晶、虎眼石等）|
| 喜金 | 白水晶、白幽灵 | 月光石、白碧玺、闪灵钻 | 火性水晶（石榴石、红发晶等）|
| 喜土 | 黄水晶、黄幽灵 | 虎眼石、黄碧玺、金发晶 | 木性水晶（绿幽灵、绿发晶等）|

**五行相生（可叠加增强）：** 金生水 → 水生木 → 木生火 → 火生土 → 土生金

**五行相克（避免同戴）：** 金克木 / 木克土 / 土克水 / 水克火 / 火克金

### 1.4 进阶：喜用神的"桥梁配戴法"

当天干地支关系复杂时，单一五行可能不够。可以用"相生链"叠加配戴：

- **日主弱金，财星木旺** → 不能只戴木（泄身），要先戴金（强身），再加一点水（金生水生木），最后加木。比例约 10:1:9
- **日主旺火，需要水来降** → 先看有没有土通关（火生土生水），如果有土可以火+土+水；如果没有土，水和火避开同戴，分时段
- **喜用神随时间变化** → 大运变了喜用神可能也变，水晶搭配要跟着调整

## 五、合盘水晶（v2 新增）

### 5.1 两人合盘推荐水晶

根据合盘分析结果，配对选择水晶：

| 合盘特征 | 推荐水晶 | 佩戴方式 | 说明 |
|----------|----------|----------|------|
| 五行互补型 | 各自戴自己的用神水晶 | 各自左手 | 各自补强，互相支撑 |
| 五行相生型 | 一方戴生另一方五行的水晶 | 生者右手、被生者左手 | 形成能量流动环路 |
| 五行冲克型 | 双方都戴茶水晶/白水晶（中性调和）| 各左手 | 中和冲突、缓冲 |
| 感情需要升温 | 两人各戴石榴石 + 粉水晶 | 男左女右 | 激活感情能量 |
| 沟通不畅 | 两人各戴海蓝宝 | 喉轮位置 | 促进表达与倾听 |
| 事业伙伴 | 两人各戴黄水晶 + 绿幽灵 | 办公桌两侧 | 增加财富与事业协同 |

### 5.2 合盘水晶禁忌

- **忌神五行相同** → 避免同戴该五行水晶，会互相放大忌神
- **水火冲克型** → 避免一人戴红发晶（火）、另一人戴黑曜石（水），会加剧对立
- **两人都喜同一五行** → 可以戴同款，但不要互相混戴对方的水晶（能量混乱）

## 七、配戴、净化与禁忌（v2 修订版）

### 7.1 配戴基本原则

| 目的 | 戴哪只手 | 原因 |
|------|----------|------|
| 招财、招桃花、提升运势 | **左手** | 左手接收宇宙能量 |
| 辟邪、化煞、防小人 | **右手** | 右手释放/排出负能量 |
| 健康、净化 | 两手皆可 | 看具体水晶功用 |

### 7.2 配戴数量

- 建议每次 **1-3种**，最多不超过5种
- 能量过多易紊乱，反而内耗
- 不同五行有相生关系的水晶可叠加（如：绿幽灵 + 紫水晶，木木相生）
- 相克五行避免同戴（如：红发晶（火）+ 白水晶（金），火克金）

### 7.3 净化方法

| 方法 | 操作 | 时间 | 注意 |
|------|------|------|------|
| 月光净化 | 农历十五放月光下 | 整夜 | 所有水晶适用，最温和 |
| 流水冲洗 | 清水持续冲洗 | 5-20分钟 | 避免温差过大，脆性水晶慎用 |
| 白水晶簇 | 放在白水晶簇上 | 数小时至一夜 | 最方便，所有适用 |
| 香熏净化 | 檀香/白鼠尾草烟熏 | 绕行三圈 | 所有适用 |
| 阳光照射 | 阳光直射 | 30分钟-1小时 | ⚠️ 紫水晶、粉水晶、绿松石会褪色！ |
| 御守盐浸泡 | 放入盐水 | 3-4小时 | ⚠️ 金属配件会被腐蚀 |
| 音频净化 | 颂钵声波 | 数分钟 | 所有适用 |

### 7.4 净化频率

- 新买水晶 → 立即净化一次再戴
- 日常配戴 → 每月 1-2 次
- 情绪激烈波动后 → 及时净化
- 医院/葬礼/负能量场所出入后 → 立即净化
- 黑曜石/黑碧玺等吸附类 → 频率加倍

### 7.5 搭配禁忌（v2 修订）

**能量相冲组合（不建议同戴）：**

| 组合 | 问题 |
|------|------|
| 红发晶 + 黑曜石 | 火水相冲、能量方向相反、互相抵消 |
| 绿幽灵 + 红纹石 | 绿主冲劲、红使沉迷情感、互相消耗 |
| 月光石 + 黑发晶 | 情感治愈与强力辟邪冲突、易情绪不稳 |
| 钛晶 + 黑曜石 | 两者能量均极强、普通人难承 |
| 石榴石 + 紫水晶（同戴于卧室）| 两者都太活跃、影响睡眠 |

**五行相克组合：**
- 火克金：红发晶/石榴石 + 白水晶（不要同戴）
- 金克木：白水晶/白幽灵 + 绿幽灵（不要同戴）
- 土克水：黄水晶/黄幽灵 + 黑曜石（不要同戴）
- 水克火：黑曜石/海蓝宝 + 石榴石（不要同戴）
- 木克土：绿幽灵 + 黄水晶（不要同戴）

### 7.6 体质注意

- 阴虚火旺、易激动者 → 慎用黄水晶、石榴石（能量偏阳）
- 体质虚弱者 → 慎用钛晶（能量过强）、Moldavite（转化太猛）
- 孕妇 → 一般建议只戴白水晶，其他慎用
- 心脏病/高血压 → 慎用红发晶、石榴石（活血）
- 失眠者 → 卧室不放石榴石、红发晶；改用紫水晶+锂云母

### 7.7 保养方法

- 不戴时放饰品盒内，避免落灰碰撞
- 避免接触化妆品、香水、洗涤剂、酒精
- 避免长期强光暴晒（部分褪色）
- 避免温度骤变（洗澡、桑拿取下）
- 定期软布轻擦保持光泽

## 九、水晶开采与伦理（v2 新增）

### 9.1 常见假冒水晶

| 水晶 | 常见假货材料 | 鉴别方法 |
|------|------------|----------|
| 捷克陨石 | 绿色玻璃 | 真品表面有天然气印纹理，轻而温感 |
| 紫水晶 | 染色玻璃/合成水晶 | 天然紫水晶颜色不均匀、有天然冰裂 |
| 黄水晶 | 加热紫水晶/染色 | 天然黄水晶色泽柔和、加热的有红调 |
| 黑曜石 | 黑色玻璃 | 真品有天然纹理、冰种有微闪 |
| 绿幽灵 | 染色/合成 | 真品内含物自然分布、颜色有层次 |
| 舒俱徕 | 染色玉髓 | 真品色泽自然、紫色有浓淡变化 |

### 9.2 伦理采购提示

- 优先选择有产地溯源的水晶
- 警惕"能量越贵越高"的营销话术——水晶价格取决于稀有度，不是能量
- 不建议购买来源不明的"古董水晶"——多数是假货
- 天然水晶有瑕疵是正常的，完美无瑕反而可疑
